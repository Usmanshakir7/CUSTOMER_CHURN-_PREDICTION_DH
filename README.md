# CUSTOMER_CHURN-_PREDICTION_DH

## Objective

The objective of this project is to predict whether a bank customer will leave the bank or continue using its services.


## Approach

* Data cleaning and removal of unnecessary columns
* Encoding categorical features such as Gender and Geography
* Exploratory Data Analysis (EDA) using graphs
* Training a classification model using Random Forest
* Evaluating model using accuracy and confusion matrix
* Feature importance analysis to identify key factors affecting churn


## Dataset Features

* CreditScore
* Geography
* Gender
* Age
* Balance
* Tenure
* EstimatedSalary
* Exited (Target variable)

---

## Results and Insights

* Females churn more as compare to males
* Geography influences churn behavior
* Random Forest provides high prediction accuracy

---

## Conclusion

This project helps banks identify customers at risk of leaving and supports better customer retention strategies using machine learning.
